##CSTM

###Entry/Exit

CSTM is entered and exited using MPC command.

|  Function  |      MPC Mode      |OP7|OP6|OP5|OP4|OP3|OP2|OP1|OP0| 
| :------: |:--------:| :-:| :-:|  :-:| :-:|  :-:| :-:|  :-:| :-:| 
| Exit CS Training Mode | CSTMX |  0|0|0|0|0|0|0|0| 
| Enter CS Training Mode | CSTMN |  0|0|0|0|0|0|0|1|
